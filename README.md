# Dolphin XR тАУ VR Game Profiles

This repository contains **VR Game Profiles** (`GameSettingsVR` `.ini` files) for use with Dolphin XR.

## Installation

To use these profiles, place the `GameSettingsVR` folder inside your Dolphin user directory.

### Steps:

1. Open Dolphin XR
2. In the top menu, click **File тЖТ Open User Folder**
3. Copy the `GameSettingsVR` folder into the directory that opens

The final structure should look like:

```
Dolphin User Folder/
тФФтФАтФА GameSettingsVR/
    тФЬтФАтФА GameID1.ini
    тФЬтФАтФА GameID2.ini
    тФФтФАтФА ...
```

## Notes

* These profiles are automatically applied when launching supported games
* Make sure Dolphin XR is closed before copying files to avoid conflicts
* You can edit `.ini` files to customize VR behavior per game
